## PlatformIO commands

### Build
ctrl + alt + b

### Upload
ctrl + alt + u

### Serial Monitor
ctrl + alt + s

### SPIFFS Data Upload
In terminal, run:

    platformio run --target uploadfs

### Clean build
platformio run --target clean
